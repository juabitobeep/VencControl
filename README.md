# VencControl 📦

App de control de vencimientos para repositores de supermercado.

---

## Cómo publicar en Vercel (paso a paso)

### Paso 1 — Crear cuenta en GitHub
1. Entrá a **github.com**
2. Hacé clic en "Sign up"
3. Completá el registro con tu email

### Paso 2 — Subir el proyecto a GitHub
1. Una vez dentro de GitHub, hacé clic en el botón verde **"New"** (arriba a la izquierda)
2. En "Repository name" escribí: `venccontrol`
3. Dejá en **Public** y hacé clic en **"Create repository"**
4. En la página siguiente, hacé clic en **"uploading an existing file"**
5. **Arrastrá toda la carpeta `venccontrol-app`** al área de carga
   - Tiene que incluir: `package.json`, carpeta `src/`, carpeta `public/`
6. Hacé clic en **"Commit changes"**

### Paso 3 — Publicar en Vercel
1. Entrá a **vercel.com**
2. Hacé clic en **"Sign up"** → elegí **"Continue with GitHub"**
3. Autorizá Vercel para acceder a tu GitHub
4. Hacé clic en **"Add New Project"**
5. Buscá y seleccioná el repositorio `venccontrol`
6. Vercel detecta automáticamente que es React — no cambies nada
7. Hacé clic en **"Deploy"**
8. Esperá 1-2 minutos ☕
9. Vercel te da una URL tipo: `https://venccontrol.vercel.app`

### Paso 4 — Instalar en el celular (como app)
**Android (Chrome):**
1. Abrí la URL en Chrome
2. Tocá los tres puntitos ⋮
3. "Añadir a pantalla de inicio"
4. ¡Listo! Aparece como app en el celular

**iPhone (Safari):**
1. Abrí la URL en Safari
2. Tocá el botón compartir 📤
3. "Añadir a pantalla de inicio"
4. ¡Listo!

Solo mandales la URL de Vercel por WhatsApp.
Cada uno la instala en su celular y tiene su lista propia.
